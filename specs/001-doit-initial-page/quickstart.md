# quickstart.md

Prerequisites: Node.js (18+), npm or pnpm.

1. Install dependencies

```bash
npm install
```

2. Run development server

```bash
npm run dev
```

3. Notes for this feature

- Tailwind: theme tokens will be used via `@theme` (configure `tailwind.config.js` if needed).
- shadcn/ui: install and configure per project conventions to scaffold modal and form components.
- date-fns: used for parsing/formatting and days-left calculations.
